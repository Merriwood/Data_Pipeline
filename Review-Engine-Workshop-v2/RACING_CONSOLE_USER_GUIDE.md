## Troubleshooting

### "The horsepower meter won't move"
- Check if you've submitted a query
- Meter only animates during processing
- If stuck, try refreshing the page

### "Follow-up suggestions aren't appearing"
- They appear after the first assistant response
- Subsequent responses always have suggestions
- If missing, try rephrasing your query

### "The status says 'Full Throttle' but nothing's happening"
- This should only last 1-2 seconds
- If it persists longer, refresh the page
- Server may need restart

### "Horsepower shows 0 even though I just queried"
- It resets immediately after processing
- This is by design (shows processing is complete)
