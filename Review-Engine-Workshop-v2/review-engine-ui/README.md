# Review Engine UI Mockup

This is a Next.js-based UI mockup for the Review Engine Dashboard.

## Setup

```bash
npm install
npm run dev
```

Visit `http://localhost:3000`

## Features Demonstrated

- ✅ KPI Dashboard (Reviews, Rating, Loyalty Points, Communities)
- ✅ Sentiment Distribution (Pie Chart)
- ✅ Review Trends (Line Chart)
- ✅ Platform Performance (Bar Chart)
- ✅ Community Insights (H3 Hexagon Clustering)
- ✅ Recent Reviews with Sentiment Analysis
- ✅ Loyalty Points Integration (Yotpo)
- ✅ White-Label Responsive Design

## Technology Stack

- **Framework**: Next.js 14 with TypeScript
- **Styling**: Tailwind CSS
- **Charts**: Recharts
- **Icons**: Lucide React
